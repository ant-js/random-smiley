# Random Smiley

Randomly get smiley emojis.

## Usage

```bash
yarn add random-smiley

# or

npm i random-smiley
```

```ts
import randomSmiley from 'random-smiley';

randomSmiley();
randomSmiley(3);
```

```js
const randomSmiley = require(random-smiley);

randomSmiley();
randomSmiley(3);
```
